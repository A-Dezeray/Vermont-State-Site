# Vermont State Website Project

A modern, responsive website showcasing Vermont's tourism, government, and popular destinations created for a WGU Web Development Project. 

## Project Information

**Course:** Western Governor's University - Front-End Web Development (D277)
**Created:** April 2024
**Last Updated:** November 2024
**Recent Improvements:** Comprehensive redesign with modern CSS, improved accessibility, and enhanced UX

``

## Features

### Pages
- **Home** - Vermont overview with Think Vermont section, Green Mountain State info, and signature events
- **Explore** - Popular destinations including state parks, historic sites, and wildlife
- **Government** - Vermont statewide officials and government resources
- **Contact** - Contact form with email validation
- **City Pages** - Detailed information about Manchester, Montpelier (capital), and Stowe

### Technical Features
- Responsive navigation with mobile hamburger menu
- Email validation with visual feedback
- Interactive hover effects
- Google Maps integration on contact page
- External links to official Vermont resources
- Mobile-responsive design

## Technologies Used

- HTML5 (Semantic markup with ARIA labels)
- CSS3 (Modern features: CSS Variables, Flexbox, Transitions, Animations)
- JavaScript (ES6+)
- Font Awesome icons
- Google Fonts (Poppins)
- Google Maps embed

## How to View

1. Open `index.html` in a web browser
2. Navigate through the site using the top navigation menu
